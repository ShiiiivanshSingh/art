#  Random Museum Art
[![Live Page](https://img.shields.io/badge/Live%20Page-GitHub%20Pages-000000?style=for-the-badge&logo=github)](https://shiiiivanshsingh.github.io/art/)

<br>

### A lightweight web app that displays random artworks from The Metropolitan Museum of Art using its public API.



### Includes Popular and Explorer modes to switch between curated masterworks and the broader collection.



### Features autoplay, dark mode, and smooth loading transitions.

### Artwork details are expandable with a direct museum catalogue link.
